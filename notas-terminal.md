Notas de Terminal - TP1
Comandos usados

1. pwd
Qué hace: Muestra la ruta completa del directorio actual.
Situación: Lo usé para saber donde me encontraba.

2. ls
Qué hace: Lista los archivos y carpetas dentro del directorio actual.
Situación: Lo usé para verificar si se habian creado los archivos que hice.

3. mkdir tp1-mi-sitio
Qué hace: Crea una  carpeta.
Situación: Lo usé para crear la carpeta del proyecto y la carpeta que contenia las imagenes.

4. cd tp1-mi-sitio
Qué hace: Cambia de directorio.
Situación: Lo usé para entrar en la carpeta principal donde guardé mi proyecto.

5. touch index.html
Qué hace: Crea un archivo vacío.
Situación: Lo usé para crear los archivos index.html, estilo.css, README.md y notas-terminal.md.

6. cat README.md
Qué hace: Muestra el contenido de un archivo en la terminal.
Situación: Lo usé para revisar que contenia el archivo README.

7. which git
Qué hace: Muestra la ruta donde está instalado Git Bash.
Situación: Lo usé para saber la ubicación donde se habia instalado Git para poder escribirla en el README.

8. code
Qué hace: Abre VS Code.
Situación: Lo usé para abrir VS Code desde el terminal.

9. clear
Qué hace: Limpia el terminal.
Situación: Lo usé para limpiar el terminal y no confundirme.

10. code --version
Que hace: Muestra la version del VS Code.
Situacion: Lo use para ver las version del VS Code y asi poder sacarle captura.
